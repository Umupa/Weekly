这一周，我开始留意城市中的光与影。那些平时被忽略的角落，在不同的光线下，呈现出截然不同的面貌。

## 清晨的街道

清晨六点的街道，阳光斜斜地照进来，在地面上投下长长的影子。咖啡店刚开门，空气中弥漫着咖啡的香气。

这是城市最安静的时刻，也是最适合拍照的时刻。

## 本周发现

### 技术

- [Astro 5.0 发布](https://astro.build) - 带来了全新的 Content Layer API
- CSS 的 `light-dark()` 函数开始被更多浏览器支持

### 设计

看到一个很棒的个人网站设计，使用了报纸式的排版风格：

- 宋体字体营造复古感
- 合适的行高让阅读更舒适
- 红色点缀增加视觉层次

### 工具

这周开始使用 **Raycast**，发现它的剪贴板历史功能特别好用。
