## 本周推荐

### 技术文章

最近在学习 [Astro](https://astro.build) 框架，发现它非常适合构建内容驱动的网站。零 JavaScript 的默认输出让页面加载速度飞快。配合 **UnoCSS** 使用，开发体验极佳。

这里是一段代码示例：

```javascript
// Astro 组件示例
---
const greeting = '你好，世界';
---

<h1>{greeting}</h1>
```

### 设计灵感

看到一个很有意思的网站设计，采用了报纸式的排版风格，给人一种复古但不过时的感觉。主要特点包括：

- 使用宋体或楷体作为主要字体
- 合适的行高和字间距
- 报头设计有仪式感
- 配色简洁大方
